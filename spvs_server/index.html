Enter a host name

A host, or website, on the Internet is identified by a host name, such as www.example.com. Host names are sometimes called domain names. Host names are mapped to IP addresses such as 255.255.255.255, but a host name and an IP address do not have a one-to-one relationship. To learn more about host names, click here!

Input formats include:

  • domain name - www.example.com
  • IP address - 255.255.255.255
  • localhost (this is your machine!)
